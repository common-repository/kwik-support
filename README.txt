== Description ==

The plugin Kwik Support is a is a simple chat script for the admin dashboard of your WordPress website. It provides a simple way for clients to communicate with the site developer and maintains a log of those communications.

Kwik Support functions best as a reference for specific site features your clients can refer to rather than relying on lengthy emails where information can be repeated. Developers can post how-to videos and links in their reply to customer questions.

== Features ==

Email notifications of new support requests
tinyMCE support
Ability to embed media in the replies
Chat log
